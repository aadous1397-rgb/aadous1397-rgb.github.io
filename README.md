# 💼 CV Personnel Interactif

## 📌 Description
Ce projet consiste en la réalisation d’un CV personnel interactif développé dans le cadre du module **Programmation Web 2**.

L’objectif est de créer une page web moderne, dynamique et responsive présentant mon profil, mes compétences et mes projets, en utilisant des technologies web actuelles.

---

## 🎯 Objectifs du projet
- Créer un CV en ligne accessible publiquement
- Appliquer les concepts de HTML, CSS, jQuery et ReactJS
- Développer une interface utilisateur interactive et agréable
- Mettre en valeur mes compétences et projets personnels

---

## 🛠️ Technologies utilisées
- HTML5 / CSS3 : structure et design responsive
- jQuery : animations, validation du formulaire et accordéon
- ReactJS : composants dynamiques pour les projets
- Font Awesome : icônes
- GitHub Pages : hébergement du site

---

## ✨ Fonctionnalités
- 🎨 Design moderne et responsive
- 📊 Barres de compétences animées avec jQuery
- 📚 Formation interactive avec accordéon
- ⚛️ Affichage des projets avec React
- 📩 Formulaire de contact avec validation
- 🔗 Liens vers GitHub, LinkedIn et Instagram
- 🎯 Animation des sections au scroll
